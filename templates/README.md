# Templates

Reusable templates for creating additional lead magnets (11-20+) and scaling the IGNITE system.

---

## Templates Available

### PROMPT-TEMPLATE.md
**Purpose:** Create new lead magnet prompts for Claude AI
**Use When:** Building magnets #11-20+
**Customization:** Fill in magnet-specific details, target audience, format

### LANDING-PAGE-TEMPLATE.html
**Purpose:** HTML structure for landing pages
**Use When:** Want consistent structure across magnets
**Customization:** Update headlines, copy, images, CTAs

### EMAIL-TEMPLATE.md
**Purpose:** Email sequence structure
**Use When:** Creating new email sequences
**Customization:** Adjust timing, tone, offer positioning

### AD-TEMPLATE.md
**Purpose:** Facebook ad copy framework
**Use When:** Creating new ad campaigns
**Customization:** Update hooks, pain points, CTAs

### VIDEO-SCRIPT-TEMPLATE.md
**Purpose:** Video script structure
**Use When:** Recording explainer videos
**Customization:** Adjust message, examples, CTA

---

## How to Use Templates

### Step 1: Identify Need
- Decide which new magnet to create (e.g., Magnet #11)
- Define target audience sub-segment
- Identify specific problem to solve
- Choose format (PDF, quiz, tool, challenge)

### Step 2: Copy Template
- Copy appropriate template file
- Rename with magnet number and name
- Save in correct folder

### Step 3: Customize
- Replace all `[PLACEHOLDER]` text
- Add magnet-specific details
- Adjust for target audience
- Maintain IGNITE brand guidelines

### Step 4: Test
- Run prompt in Claude (if prompt template)
- Review generated output
- Verify brand alignment
- Refine as needed

### Step 5: Implement
- Follow standard implementation process
- Track performance
- Document lessons learned

---

## Template Maintenance

Update templates when:
- Brand guidelines change
- Better structures discovered through testing
- Platform requirements evolve
- Performance data reveals improvements

**Version Control:**
- Keep original templates as v1
- Create v2/v3 as improvements made
- Document changes in template header

---

## Creating Custom Templates

As you scale beyond 20 magnets:

1. **Identify patterns** in high-performing magnets
2. **Extract common elements** (structure, copy frameworks, design patterns)
3. **Create new template** documenting the pattern
4. **Test template** by creating 2-3 magnets with it
5. **Refine based on results**
6. **Add to this folder** with clear documentation

**Templates accelerate creation while maintaining quality. Use them to scale faster.**
