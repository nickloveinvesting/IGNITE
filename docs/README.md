# Documentation Overview

This folder contains all essential documentation for the IGNITE Lead Magnets system.

## Files in This Folder

### BRAND_GUIDELINES.md
Complete brand specifications for IGNITE including:
- Color palette with hex codes
- Typography specifications
- Voice and tone guidelines (Toby Potter style)
- Visual identity standards
- Usage examples

**When to use:** Before creating any content, designing artifacts, or writing copy.

---

### EXECUTION_GUIDE.md
Step-by-step instructions for running all 10 lead magnet prompts:
- Parallel execution setup
- Sequential execution process
- Troubleshooting prompt generation
- Organizing downloaded artifacts
- Quality assurance checks

**When to use:** When ready to generate lead magnet artifacts using Claude AI.

---

### IMPLEMENTATION_ROADMAP.md
Week-by-week deployment plan for all 10 magnets:
- Timeline from generation to launch
- Resource allocation
- Priority sequencing
- Milestone tracking
- Success metrics by phase

**When to use:** Planning implementation timeline and resource allocation.

---

### RESEARCH_FRAMEWORK.md
Multi-agent research methodology integrated into prompts:
- Competitive intelligence approach
- Audience psychology research
- Platform optimization research
- Web search query templates
- Research synthesis methods

**When to use:** Understanding how research informs artifact generation and when creating new prompts.

---

### QUALITY_CHECKLIST.md
Pre-launch verification checklist for each lead magnet:
- Brand alignment checks
- Technical functionality tests
- Copy and design reviews
- Integration verifications
- Performance benchmarks

**When to use:** Before launching any lead magnet to ensure quality standards.

---
