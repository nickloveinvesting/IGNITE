## Team Structure

### Core Roles

**AI Prompt Executor**
- Runs all prompts in Claude
- Downloads and organizes artifacts
- Quality checks outputs
- Time: 4-6 hours (Week 1)

**Landing Page Builder**
- Builds pages in Leadpages
- Implements HTML from artifacts
- Optimizes for mobile
- Time: 20-30 hours (Weeks 1-3)

**Email Marketing Manager**
- Sets up sequences in ESP
- Configures automation
- Monitors deliverability
- Time: 15-20 hours (Weeks 1-3)

**Paid Ads Manager**
- Creates Facebook campaigns
- Manages budgets
- Optimizes performance
- Time: 10-15 hours/week (ongoing)

**Designer**
- Designs PDFs and graphics
- Creates ad visuals
- Ensures brand consistency
- Time: 25-35 hours (Weeks 1-3)

**Video Producer**
- Records video scripts
- Edits and optimizes
- Publishes and embeds
- Time: 20-30 hours (Weeks 1-4)

**Project Manager**
- Coordinates team
- Tracks progress
- Troubleshoots blockers
- Reportsto stakeholders
- Time: 5-10 hours/week (ongoing)

## Collaboration Guidelines

### Communication Tools
- **GitHub:** Repository, documentation, version control
- **Slack:** Daily communication (#ignite-lead-magnets channel)
- **Asana/Trello:** Task management, timeline tracking
- **Google Sheets:** Metrics dashboard
- **Loom:** Video walkthroughs, training

### Daily Standup (Async in Slack)
- What did you complete yesterday?
- What are you working on today?
- Any blockers?
- 15 minutes max

### Weekly Review (60-minute video call)
- Review metrics from launched magnets
- Discuss optimization opportunities
- Align on priorities for coming week
- Celebrate wins, troubleshoot challenges

### File Naming Convention
- Landing pages: `[magnet-number]-[magnet-name]-landing-page.html`
- Emails: `[magnet-number]-[magnet-name]-email-[sequence-number].md`
- Ads: `[magnet-number]-[magnet-name]-ad-[variation].md`

## Onboarding New Team Members

1. **Access Setup**
   - GitHub repository access
   - Slack channel invite
   - Asana/Trello board access
   - Platform credentials (Leadpages, ESP, etc.)

2. **Documentation Review**
   - Read main README.md
   - Read `/docs/BRAND_GUIDELINES.md`
   - Read role-specific guides

3. **First Tasks**
   - Shadow existing team member
   - Complete one magnet end-to-end
   - Ask questions in Slack
