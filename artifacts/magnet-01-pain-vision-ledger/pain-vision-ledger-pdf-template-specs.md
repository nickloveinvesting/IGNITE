## DESIGN ELEMENTS & STYLING

### Colors (Apply Throughout):

**Primary Colors**:
- Charcoal: #121212 (text)
- Fire Gradient: #C84A00 → #FF6A00 (accents, headers)
- White: #FFFFFF (background)

**Secondary Colors**:
- Slate: #94a3b8 (secondary text)
- Light Gray: #D1D5DB (lines, prompts)
- Background Box: #F9FAFB (subtle backgrounds)

### Typography:

**Font Family**: Inter (Google Fonts, free)
- Download: https://fonts.google.com/specimen/Inter
- Weights needed: Regular (400), Semi-Bold (600), Bold (700)
- Fallback: -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif

**Font Sizes**:
- Main Title: 36pt
- Section Headers: 24pt
- Subheaders: 16-20pt
- Body Text: 11-14pt
- Prompts/Small Text: 9pt

**Line Heights**:
- Headers: 1.2
- Body text: 1.6
- Instructions: 1.4

### Spacing:

- Generous margins: 0.75" all sides
- Ample white space between sections
- 1" between writing lines (comfortable handwriting space)
- 0.5" gutter between columns
- Consistent padding in boxes: 0.15"

### Print Considerations:

**Color Mode**: CMYK for professional printing, RGB for digital
**Resolution**: 300 DPI minimum
**Bleed**: 0.125" bleed if printing professionally (extend background colors)
**Safe Zone**: Keep critical text 0.25" inside margins

**Grayscale Printing**:
- Ensure fire gradient colors convert well to grayscale
- Test print in black & white (should still be readable)
- Use solid colors if gradient doesn't print well

**Font Embedding**:
- Embed all fonts in PDF (ensure compatibility)
- Use standard fonts (Inter widely supported)

## FILLABLE FORM OPTIONS

### Option A: Print & Handwrite (Recommended)

**Why Recommended**:
- Research shows handwriting increases neural activation
- Physical act of writing enhances commitment
- More personal, emotionally resonant

**Implementation**:
- Provide as standard PDF (non-fillable)
- Include clear writing lines
- Optimize for print quality

### Option B: Fillable PDF (Alternative)

**Why Optional**:
- Some users prefer digital completion
- Easier to edit/revise
- Can be saved and re-read on devices

**Implementation** (if fillable):
- Use Adobe Acrobat Pro to add form fields
- Text fields for each of the 5 lines (left and right columns)
- Text field for signature (or signature field)
- Date field (auto-populated option)
- Checkbox fields for 28-day tracker

**Form Field Specs**:
- Font: Inter Regular, 12pt
- Color: #121212
- Border: None or very subtle
- Align with existing lines

## FILE EXPORT & DELIVERY

### Export Settings (Canva):
1. Download as PDF (Print)
2. Ensure "Bleed" is included (if professional printing)
3. Flatten transparency
4. Embed fonts
5. High resolution (300 DPI)

### Export Settings (Adobe InDesign):
1. File > Export > Adobe PDF (Print)
2. Preset: High Quality Print
3. Compatibility: Acrobat 5 (PDF 1.4) or higher
4. Embed all fonts
5. Marks and Bleeds: Include bleed if needed

### File Size Optimization:
- Target: Under 5MB (for easy email delivery)
- Compress images if needed (while maintaining 300 DPI)
- Optimize PDF in Acrobat (File > Save As Other > Optimized PDF)

### Delivery Methods:
- **Email Attachment**: Attach directly to Email 1 (if under 5MB)
- **Download Link**: Host on Dropbox, Google Drive, or server
