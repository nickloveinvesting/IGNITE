# NO EXCUSES MANDATE BUILDER WORKSHEET
## Complete Design Specifications & Production Guide

## OVERVIEW

**Format:** 1-page fillable PDF worksheet
**Size:** 8.5" x 11" (Letter size)
**Orientation:** Portrait
**File Size Target:** Under 5MB
**Software:** Adobe InDesign or Illustrator (export to fillable PDF)
**Fonts:** Inter (Regular 400, Semi-Bold 600, Bold 700)
**Colors:** Fire Orange (#FF4D00), Charcoal (#2C3539), White (#FFFFFF)

**Margins:**
- Top: 0.5"

## ACCESSIBILITY REQUIREMENTS

**PDF Accessibility Features:**
- Tagged PDF structure for screen readers
- Alt text for IGNITE logo: "IGNITE Personal Transformation Logo"
- Logical reading order: Top to bottom, left to right
- Form field tab order: Excuse #1 → Cost #1 → Mandate #1 → Excuse #2 → Cost #2 → Mandate #2 → Excuse #3 → Cost #3 → Mandate #3 → Signature → Date
- Sufficient color contrast ratios:
  - Fire Orange on White: 3.36:1 (AA for large text)
  - Charcoal on White: 12.63:1 (AAA)
  - White on Charcoal: 12.63:1 (AAA)
  - White on Fire Orange: 3.36:1 (AA for large text)

## EXPORT SETTINGS

### PDF Export (Adobe InDesign/Illustrator)

**General:**
- Compatibility: Acrobat 8 (PDF 1.7) or higher
- Optimize for: Web viewing (Fast Web View enabled)
- Include: Bookmarks, Hyperlinks, Interactive Elements

**Compression:**
- Color Images: JPEG, Maximum quality, 300 PPI
- Grayscale Images: JPEG, Maximum quality, 300 PPI
- Monochrome Images: CCITT Group 4, 1200 PPI
- Compress Text and Line Art: Yes

**Marks and Bleeds:**
- Trim Marks: No
- Bleed: 0.125" all sides (for full-bleed elements like header/footer)

**Output:**
- Color Conversion: Convert to Destination (Preserve Numbers)
- Destination: sRGB IEC61966-2.1
- Profile Inclusion Policy: Don't Include Profiles

**Security:**
- No password protection (free lead magnet)
- Allow printing: Yes
- Allow changes: Filling in form fields only
- Allow copying: Yes
- Allow screen readers: Yes

## PRINT VERSION SPECIFICATIONS

**If creating a separate printable version:**

- Same design as fillable PDF
- Remove interactive form fields
- Replace with:
  - Horizontal lines for text entry (3-4 lines per field)
  - Blank line for signature (with "X" indicator)
  - Blank line for date
- Line style: 1pt solid #CCCCCC
- Line spacing: 0.3" between lines
- Ensure sufficient space for handwriting

## USAGE INSTRUCTIONS FOR USERS

**To include in delivery email:**

**DIGITAL USE:**
1. Download the PDF to your device
2. Open in Adobe Reader (free) for best results
3. Click on each field to type your responses
4. Save your completed worksheet
5. Read daily (set a reminder!)

**PRINT USE:**
1. Download the printable version
2. Print on standard letter-size paper (8.5" x 11")
3. Use a pen to fill in your responses
4. Sign and date at the bottom
5. Post somewhere visible (bathroom mirror, desk, fridge)

**BEST PRACTICE:**
- Complete in one sitting (15-20 minutes)
- Be brutally honest with yourself
- Make mandates specific and measurable
- Read your mandates every morning for 28 days
- Track your progress (check off days completed)

## CONVERSION OPTIMIZATION NOTES

**Strategic Design Choices:**

1. **Visual Hierarchy**: Orange headers draw eye to mandate column (desired outcome)
2. **Example Prompts**: Reduce friction, show what "good" looks like
3. **Commitment Section**: Creates psychological contract (signing = binding)
4. **28-Day Instruction**: Specific timeframe (not vague "forever")
5. **"No Excuses" in footer**: Reinforces core message
6. **White Space**: Prevents overwhelm, makes worksheet feel completable
7. **3 Excuses Only**: Not 5, not 10—achievable number reduces resistance
